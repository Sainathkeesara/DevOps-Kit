# Index: Topics

A categorized guide to the contents of devops-kit.

## Setup Guides (`docs/setup-guides/`)
- Step-by-step instructions for installing and configuring tools and platforms

## How-To Guides (`docs/how-to/`)
- Task-oriented tutorials for specific DevOps workflows

## Concepts (`docs/concepts/`)
- Explanations of DevOps principles, patterns, and best practices

## Troubleshooting (`docs/troubleshooting/`)
- Symptom → cause → resolution patterns for common issues

## Runbooks (`docs/runbooks/`)
- Operational procedures for incident response and maintenance

## Reference (`docs/reference/`)
- Quick-reference material, cheatsheets, and comparisons

## Scripts (`scripts/`)
- Reusable automation scripts in bash, Python, and PowerShell
  - `bash/` – Linux/macOS shell scripts
  - `python/` – Cross-platform Python utilities
  - `powershell/` – Windows PowerShell modules
  - `lib/` – Shared helper libraries
  - `examples/` – Example script demonstrations

## Snippets (`snippets/`)
- Copy-and-paste code fragments for common tasks

## Templates (`templates/`)
- Starter skeletons for projects, Dockerfiles, Kubernetes manifests, Terraform modules, and documentation

## Tools Roadmap (`CHANGELOG.md` and quick links)
- Track planned and in-progress toolkits
